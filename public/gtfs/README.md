# GTFSファイル配置ディレクトリ

このディレクトリに、滋賀県の公共交通オープンデータ（GTFS形式）のファイルを配置してください。

## 必要なファイル

以下のGTFSファイルを配置してください：

- `stops.txt` - 停留所情報
- `stop_times.txt` - 各停留所の到着・出発時刻
- `trips.txt` - 運行パターン情報
- `routes.txt` - 路線情報
- `calendar.txt` - 運行カレンダー

## ファイルの取得方法

1. 滋賀県の公共交通オープンデータポータルサイトからGTFSデータをダウンロード
2. ZIPファイルを解凍
3. 必要なファイルをこのディレクトリに配置

## ファイル形式

GTFSファイルはCSV形式で、UTF-8エンコーディングである必要があります。

## 注意事項

- ファイルは必ずUTF-8エンコーディングで保存してください
- ファイル名は正確に上記の通りにしてください（大文字小文字を区別）
- ファイルが存在しない場合、エラーが発生します
